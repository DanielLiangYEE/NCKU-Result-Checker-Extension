# 🚀 115 碩士班考試 查榜助手 (v1.0)

[![Chrome Extension](https://img.shields.io/badge/Platform-Chrome%20Extension-blue.svg)](https://developer.chrome.com/docs/extensions/mv3/getstarted/)
[![Internal Font](https://img.shields.io/badge/Font-Noto%20Sans%20TC-red.svg)](https://fonts.google.com/specimen/Noto+Sans+TC)

這是一款專為碩士班考生設計的 Chrome 擴充功能，旨在簡化繁瑣的查榜流程。透過直覺的介面與自動化腳本，讓您在放榜時刻能一鍵快速切換至目標科系的查榜頁面，節省手動輸入與選取的寶貴時間。

## ✨ 功能亮點

- **🎓 多校支援**：整合 **成功大學 (NCKU)**、**台灣大學 (NTU)**、**中山大學 (NSYSU)** 的自動化查詢。
- **⚡ 一鍵跳轉**：自動填寫報名序號、選擇考試類別並自動點選「查詢」。
- **❤️ 收藏清單**：您可以將所有報考的科系加入關注列表，隨時點擊查詢。
- **🎨 精緻介面**：採用 Apple 風格的純淨設計語法，提供極致的視覺舒適度。
- **🔡 本地化顯示**：內建 **Noto Sans TC (思源黑體)** 變體字體，保證在任何環境下皆有完美的繁體中文顯示效果。
- **🔒 隱私與效能**：100% 本地運行，不儲存任何個人隱私，亦不向第三方伺服器發送請求（字體已完全本地化）。

## 🛠️ 安裝方法 (開發者模式)

1. **下載原始碼**：點擊右上方 `Code` -> `Download ZIP` 並解壓縮，或使用 `git clone`。
2. **開啟擴充功能頁面**：在 Chrome 網址列輸入 `chrome://extensions/`。
3. **啟用開發者模式**：開啟頁面右上角的「開發者模式」開關。
4. **載入擴充功能**：點擊左上角「載入解壓縮後的擴充功能」，並選取本專案的根目錄資料夾。

## 📖 使用說明

1. 點擊瀏覽器工具列中的 **「查榜助手」** 圖示。
2. 在下拉選單中選取您的 **目標學校**。
3. 在搜尋框中輸入 **科系代碼** 或 **名稱關鍵字** (例如：`41101` 或 `中國文學`)。
4. 點擊 **「新增」** 將其加入您的關注清單。
5. 點擊清單中的 **「查詢」**，助手將自動為您開啟並點選相關網頁。

## 📁 專案架構

```text
├── manifest.json        # 擴充功能核心設定 (MV3)
├── popup.html           # 助手彈窗介面
├── popup.css            # 緊緻化、Apple 風格樣式
├── popup.js             # 介面邏輯與搜尋引擎
├── data.js              # 跨校資料庫整合
├── background.js        # 背景自動化跳轉邏輯
├── fonts/               # 內建 Noto Sans TC 本地字體
├── ncku_data.js         # 成大資料集
├── ntu_data.js          # 台大資料集
└── nsysu_data.js        # 中山資料集 (支援兩階段跳轉)
```
