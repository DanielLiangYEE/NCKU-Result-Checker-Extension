# 115 碩士班考試 查榜助手

[![Chrome Extension](https://img.shields.io/badge/Platform-Chrome%20Extension-blue.svg)](https://developer.chrome.com/docs/extensions/mv3/getstarted/)
[![Internal Font](https://img.shields.io/badge/Font-Noto%20Sans%20TC-red.svg)](https://fonts.google.com/specimen/Noto+Sans+TC)

這是一款專為碩士班考生設計的 Chrome 擴充功能，旨在優化並自動化各校碩士班考試累積的查榜流程。透過後台腳本與遠端組態同步技術，確保考生能掌握最新的放榜動態，並實現一鍵跳轉至目標科系查詢結果。

## 主要功能

- **多校自動化查詢**：整合台灣大學 (NTU)、成功大學 (NCKU)、中央大學 (NCU)、中山大學 (NSYSU) 等校之自動化查詢腳本，並預留政治大學 (NCCU) 與中正大學 (CCU) 之支援結構。
- **遠端動態組態**：透過遠端組態系統即時同步各校查榜系統之開放狀態與即時公告。當校方系統維修或尚未開放時，擴充功能將自動標記並進行操作限制。
- **即時公告系統**：整合跑馬燈公告功能，支援重要訊息提示、更新連結跳轉與已讀管理。
- **高效系所檢索**：支援系所名稱與代碼之即時模糊搜尋，並自動清理原始資料中冗餘的編碼與符號，提供簡潔直觀的檢索體驗。
- **關注清單管理**：提供持久化的關注列表，支援跨校科系收藏、一鍵批次查詢與動畫視覺反饋。
- **後台靜默處理**：查詢流程於後台視窗執行，自動完成表單填寫與提交動作，完成後自動切換至結果分頁。
- **隱私安全規範**：完全本地端運行數據儲存，不涉及個人隱私收集，所有外部請求僅用於同步遠端組態與公告。

## 安裝步驟

1. **取得原始碼**：下載專案 ZIP 檔案並解壓縮，或使用 Git 進行複製。
2. **開啟擴充功能管理頁面**：於 Chrome 網址列輸入 chrome://extensions/。
3. **啟用開發者模式**：開啟頁面右上角之開發者模式開關。
4. **載入擴充功能**：點擊「載入未封裝項目」並選擇本專案資料夾。

## 使用流程

1. 點擊工具列之查榜助手圖示開啟介面。
2. 於下拉選單切換至目標學校，系統會自動載入該校系所快取。
3. 於搜尋框輸入系所名稱或代碼關鍵字。
4. 點擊「新增」按鈕將科系加入關注列表。
5. 點擊列表項中之「查詢」按鈕，系統將自動啟動背景流程完成查詢。
