# NCKU Result Checker Helper (成大查榜助手) 

這是一個專為成大考生設計的 Chrome 擴充功能，旨在解決複數系所備取進度查詢繁瑣的問題。

## ✨ 主要功能 (Features)
- **多系所記憶**：自動儲存感興趣的系所，下次開啟免重複選取。
- **一鍵導向**：直接開啟新分頁並自動填寫、點擊查詢。
- **隱私安全**：所有資料僅存在個人瀏覽器 (Local Storage)，不外傳。

## 🛠️ 安裝教學 (Installation)
1. 下載本專案並解壓縮。
2. 開啟 Chrome 瀏覽器，進入 `chrome://extensions/`。
3. 開啟右上角 **「開發者模式 (Developer Mode)」**。
4. 點擊 **「載入解壓縮擴充功能 (Load unpacked)」**，選擇此資料夾。
5. 將插件 **固定 (Pin)** 在工具列，即可開始使用。
